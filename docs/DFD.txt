Diagrama de Fluxo de Dados (DFD) - Sistema de Gerenciamento de Alunos

Entidades Externas:
1. Administrador
2. Aluno

Processos:

1. Gerenciamento de Alunos
   - Entrada: Dados do aluno (Administrador)
   - Saída: Informações do aluno (Administrador)
   - Armazenamento: Banco de dados (Tabela Alunos)

2. Gerenciamento de Matrículas
   - Entrada: Dados da matrícula (Administrador)
   - Saída: Confirmação de matrícula (Administrador)
   - Armazenamento: Banco de dados (Tabela Matrículas)

3. Gerenciamento de Notas
   - Entrada: Notas dos alunos (Administrador)
   - Saída: Boletim escolar (Administrador, Aluno)
   - Armazenamento: Banco de dados (Tabela Notas)

Fluxos de Dados:

1. Administrador -> Gerenciamento de Alunos -> Banco de dados
2. Administrador -> Gerenciamento de Matrículas -> Banco de dados
3. Administrador -> Gerenciamento de Notas -> Banco de dados
4. Banco de dados -> Gerenciamento de Alunos -> Administrador
5. Banco de dados -> Gerenciamento de Matrículas -> Administrador
6. Banco de dados -> Gerenciamento de Notas -> Administrador/Aluno

Nota: Este é um modelo textual. Um diagrama visual seria criado em uma ferramenta como Draw.io, Lucidchart, etc.
