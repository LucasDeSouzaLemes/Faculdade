Modelo Entidade-Relacionamento (MER) - Sistema de Gerenciamento de Alunos

Entidades:

1. ALUNO
   - id (PK)
   - nome
   - data_nascimento
   - endereco
   - telefone

2. MATRICULA
   - id (PK)
   - aluno_id (FK)
   - data_matricula
   - status

3. NOTA
   - id (PK)
   - aluno_id (FK)
   - valor
   - data_avaliacao
   - observacao

Relacionamentos:

1. ALUNO possui MATRICULA (1:N)
2. ALUNO possui NOTA (1:N)
